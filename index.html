Element 34's PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover element-34.github.com/pear
Listing available packages: 
pear remote-list -c element-34
Installing a package: 
pear install element-34/package_name
Installing a specific version/stability: 
pear install element-34/package_name-1.0.0
pear install element-34/package_name-beta
Receiving updates via a feed: 
http://element-34.github.com/pear/feed.xml
